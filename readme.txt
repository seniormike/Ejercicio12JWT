*** IMPORTANTE ***
En la base de datos se encuentran 3 usuarios con roles diferentes:
El rol admin permite tanto publicar como obtener mensajes.
El rol viwere solamente permite obtener mensajes.
El rol publisher solamente puede publicar mensajes.

En la base de datos se encuentran los usuarios contraseñas y roles. Se utiliza una base de datos en la nube Mongo Atlas.
Se encuentran las collecciones postman para realizar las pruebas en la carpeta collections.
